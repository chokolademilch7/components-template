Quick Start
```
npx degit chokolademilch7/components-template [names]
```

```
npm i
or
yarn
```
```
npm run dev
or
yarn dev
```

localhost  
http://localhost:5000

src/components

src/index.js
作成した`hoge.js`をインポートしてください

public/index.html
作成したコンポーネントの名前のタグを書いてください  
`<hoge-button></hoge-button>`
